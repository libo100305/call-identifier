## 🔧 常见问题

### Q1: 浏览器中没有看到应用？

**A**: 请检查：
1. 浏览器地址是否为 `http://localhost:63285`
2. Flutter 是否正在运行
3. 查看终端是否有错误信息

### Q2: 页面显示空白或错误？

**A**: 
1. 刷新浏览器页面
2. 查看浏览器控制台（F12）的错误信息
3. 查看 Flutter 终端的输出

### Q3: 如何停止应用？

**A**: 
在运行 Flutter 的终端按 `Ctrl + C`

### Q4: 如何重新运行？

**A**: 
```bash
flutter run -d chrome
```
