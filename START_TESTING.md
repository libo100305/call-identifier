## ✅ 选择 1：使用 Android 模拟器（推荐新手）

**优点：** 无需真机，随时可测试
**缺点：** 首次启动较慢，需要电脑性能较好

### 启动步骤：

1. **打开 Android Studio**
2. **启动模拟器**：
   - 点击工具栏的 Device Manager
   - 找到 "Pixel 4a" 模拟器
   - 点击 ▶️ 启动按钮
   - 等待模拟器完全启动（约 1-2 分钟）

3. **运行应用**：
   ```bash
   cd C:\Users\admin\Desktop\xmwj\dianhua
   flutter run
   ```

### 或者使用命令行启动：

```bash
# 1. 启动模拟器
flutter emulators --launch Pixel_4a

# 2. 等待 2 分钟让模拟器完全启动

# 3. 运行应用
flutter run
```

## ✅ 选择 2：连接 Vivo 真机测试

**优点：** 真实性能，完整测试
**缺点：** 需要设置开发者选项

### 快速连接步骤：

1. **启用开发者模式**：
   - 设置 → 关于手机 → 连续点击"版本号"7 次

2. **开启 USB 调试**：
   - 设置 → 系统管理 → 开发者选项
   - 开启"USB 调试"

3. **连接 USB**：
   - 用数据线连接手机和电脑
   - 手机上点击"允许 USB 调试"
   - 选择"传输文件"模式

4. **验证连接**：
   ```bash
   adb devices
   ```
   看到设备 ID 表示成功

5. **运行应用**：
   ```bash
   flutter run
   ```
